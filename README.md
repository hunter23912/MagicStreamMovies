# 一个个人电影推荐网站

> 记录个人的`react`和`gin`学习历程

### 网站架构

- 前端：`react`框架
- 后端：`gin`框架

### `react`框架重点

1. 组件化和组合，`react`的核心就是拼组件，小组件组合成页面。
2. `props`负责传参，父组件把数据和函数传给子组件，子组件通过它接收外部信息。
3. `state`负责状态，页面里会变化的数据一般放进 `state`，由状态驱动界面更新。
4. 单向数据流，数据通常从父组件流向子组件，子组件想改父组件数据时，通过回调函数上抛。
5. 条件渲染和列表渲染，页面显示什么、显示几项，主要靠 `if`、三元表达式、`map` 这些 JavaScript 写法控制。
6. `useEffect`处理副作用，像请求数据、订阅事件、操作 DOM、定时器这类“渲染之外的事”都放这里。
7. 状态提升和 `hooks` 复用，多个组件共享数据时把状态提到共同父组件，重复逻辑则抽成自定义 `hooks`。
8. 不可变更新和 `key`，更新数组对象时不要直接改原值，列表渲染时要写好 `key`，否则容易出 bug。
